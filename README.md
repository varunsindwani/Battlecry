##Get started

1. **Create a Gatsby site.**
```sh
gatsby new my-webpage https://github.com/varunsindwani/Battlecry-Starter
```

```sh
2. **Start developing.**
cd my-webpage
gatsby develop
```

```sh
3. **Open the source code and start editing!**
Your site is now running at http://localhost:8000!
```
